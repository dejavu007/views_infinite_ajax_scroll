This module is to integrate the infinite ajax scroll plugin ( http://infiniteajaxscroll.com ) with views as pager.


-- REQUIREMENTS --
* Libraries module - https://www.drupal.org/project/libraries .
* Need to add "jquery-ias.min.js" file from "http://infiniteajaxscroll.com/download.html" to libraries. See INSTALLATION.


-- INSTALLATION --
1) Download "jquery-ias.min.js" file from "http://infiniteajaxscroll.com/download.html". Place it in 'sites/all/libraries/infinite-ajax-scroll'. Make sure libraries module is installed.
2) Install this module as usual, see http://drupal.org/node/895232 for further information.
3) Go to the views for which it need to enabled. Under pager section choose the pager as "Infinite Ajax Scroll". You can change the options given to Infinite Ajax Scroll plugin in the pager settings if required. Currently it support only few options.
